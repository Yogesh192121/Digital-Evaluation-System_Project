🎓 Digital Evaluation System

Digital Evaluation System is a web-based application developed using Python and Flask to simplify the process of managing student examinations, evaluations, and result generation. It provides an efficient and automated platform for handling academic data, reducing manual effort, and improving accuracy in evaluation.

The system includes features such as student management, exam evaluation, and report generation with a user-friendly interface built using HTML, CSS, and JavaScript. It uses SQLite as the database, making it lightweight, easy to deploy, and suitable for academic institutions or student-level projects.

🚀 Features

📝 Student Management
📊 Exam Evaluation
📈 Result Generation
📂 Report Management
🔐 Secure Data Handling

🛠️ Tech Stack

Backend: Python (Flask)
Frontend: HTML, CSS, JavaScript
Database: SQLite
Version Control: Git & GitHub

⚙️ Installation & Setup

1. Clone the repository
```id="v3sh64"

```
2. Navigate to project folder
```id="7rf6y5"
cd Digital-Evaluation-System_Project
```
3. Install dependencies
```id="g4ph1r"
pip install -r requirements.txt
```
4. Run the application
```id="kw3z8o"

```
5. Open in browser
```id="z2v9hq"
http://127.0.0.1:5000/
```
